# 逆运动学求解器改进说明

## 问题描述

原有代码在使用雅可比阻尼最小二乘法（DLS）进行逆运动学求解时，当大小臂夹角大于90度（机械臂接近伸直状态）时，会出现目标位置不准确的问题。

## 问题根源

1. **多解问题**：对于6轴机械臂的给定TCP位置，通常存在两组不同的关节角度解：
   - **肘部向下（elbow down）**：大小臂夹角 < 90°，q3 为负值
   - **肘部向上（elbow up）**：大小臂夹角 > 90°，q3 为正值

2. **初始值单一**：原代码的 `analytic_seed` 函数只生成肘部向下配置的初始值，当目标位置需要肘部向上配置时，DLS算法可能收敛到错误的局部最优解。

3. **DLS参数固定**：固定的阻尼和步长参数在某些位形下可能不是最优的。

## 改进方案

### 1. 支持肘部配置选择

修改了 `analytic_seed` 函数，添加 `elbow_down` 参数：
- `elbow_down=True`：生成肘部向下配置的初始值（q3 为负）
- `elbow_down=False`：生成肘部向上配置的初始值（q3 为正）

```python
def analytic_seed(target_xyz, current_q_model, elbow_down=True):
    # ...
    if elbow_down:
        q3 = -math.acos(cos_q3)  # 肘部向下
    else:
        q3 = math.acos(cos_q3)   # 肘部向上
```

### 2. 智能求解策略

`solve_xyz` 方法现在支持三种模式：

- **肘部向下模式**（`elbow_down=True`）：优先使用肘部向下配置，适合试管抓取等场景
- **肘部向上模式**（`elbow_down=False`）：优先使用肘部向上配置
- **自动模式**（`elbow_down=None`）：同时尝试两种配置，自动选择误差最小的解

### 3. 增强的后备策略

当指定配置求解失败时，系统会：
1. 首先尝试从当前姿态开始求解
2. 如果仍失败，尝试另一种肘部配置作为后备
3. 最终选择误差最小的解

### 4. 优化的DLS参数

- 降低了默认阻尼值（0.03），提高收敛速度
- 增加了最大步长（0.08），加快迭代
- 添加了 `_try_multiple_dls` 方法，可尝试多组参数

### 5. 大小臂夹角警告

当检测到大小臂夹角大于90度时，会输出警告信息，提示用户机械臂接近伸直状态。

## 使用方法

### 命令行使用

```bash
# 默认肘部向下配置（适合试管抓取）
python coordinate_move_dls.py 0.25 0.0 0.20

# 使用肘部向上配置
python coordinate_move_dls.py 0.25 0.0 0.20 --elbow up

# 自动选择最优配置
python coordinate_move_dls.py 0.25 0.0 0.20 --elbow auto

# 要求工具轴竖直向下
python coordinate_move_dls.py 0.25 0.0 0.20 --tool-down
```

### 编程接口

```python
from coordinate_move_dls import CoordinateArmController

controller = CoordinateArmController(
    serial_port="/dev/ttyACM0",
    dry_run=True,  # 离线模式
    allow_unmeasured_limits=True,
)

# 移动到目标位置，使用肘部向下配置
controller.move_to_xyz(
    target_xyz=[0.25, 0.0, 0.20],
    joint_speed=0.15,
    tool_down=True,
    require_confirmation=False,
    elbow_down=True,  # 肘部向下配置
)
```

## 注意事项

1. **试管抓取场景**：推荐使用 `--elbow down`（默认），保持肘部向下配置
2. **接近伸直状态**：当大小臂夹角接近180°时，机械臂处于奇异位形附近，建议调整目标位置
3. **自动模式**：当不确定使用哪种配置时，可以使用 `--elbow auto` 让系统自动选择

## 文件变更

- `coordinate_move_dls.py`：主要改进文件
  - 修改了 `analytic_seed` 函数，支持肘部配置选择
  - 添加了 `analytic_seed_dual` 函数，同时生成两种配置
  - 修改了 `solve_xyz` 方法，实现智能求解策略
  - 添加了 `_dls_solve_with_params` 和 `_try_multiple_dls` 辅助方法
  - 添加了命令行参数 `--elbow`

- `test_ik_improved.py`：新增测试脚本，用于验证改进效果
